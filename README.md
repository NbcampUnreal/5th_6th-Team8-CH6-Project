## ⚙️ 프로젝트 핵심 기술 (Technical Highlights)

> 🚀 게임 시스템을 구성하는 핵심 기술 및 구조

---

### ⚔️ Gameplay Ability System (GAS)

- 데이터 기반 어빌리티 주입과 서버 중심 초기화 구조로 확장성과 네트워크 안정성을 동시에 확보

---

### 🌐 Steam Session & GameLift

- Steam 기반 멀티플레이 세션 구성  
- GameLift를 활용한 서버 운영 환경 고려  

---

### 🔄 Seamless Travel

- 맵 전환 시 로딩 없이 자연스럽게 이동  
- 게임 흐름이 끊기지 않는 플레이 경험 제공  

---

### 🧠 SubSystem 구조 (GameInstance / World)

-  엔진 Lifecycle에 맞춘 Subsystem 아키텍처를 도입하여 전역 시스템의 안정성과 확장성을 확보 

---

### 🎮 미니게임 아키텍처

- 각 미니게임을 독립적으로 구성  
- 새로운 게임 추가 시 기존 코드 수정 최소화  

---

### 🎒 인벤토리 구조

- ItemDefinition 기반 데이터 주도 구조로 아이템 생성부터 능력 부여, 제거까지 일관된 생명주기 관리 구현  

---

### 👁️ 관전 기능

- 사망 후 다른 플레이어를 관전 가능  
-  ViewTarget 기반 시점 전환으로 자연스러운 관전 경험 제공 

---

### 🧪 **개발자 기능 Subsystem**  

- 빠른 테스트와 디버깅을 위해 게임 속도 조절, 미니게임 강제 제어 등 개발자 전용 기능을 Subsystem으로 구현
